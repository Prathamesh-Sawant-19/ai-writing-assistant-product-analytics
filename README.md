# AI Writing Assistant Product Analytics & A/B Testing

## Project Overview

This project simulates product analytics for an AI Writing Assistant platform using Python, SQL-style analysis logic, and Power BI.

The objective was to analyze:
- User retention behavior
- Prompt quality performance
- Feedback distribution
- A/B testing impact on onboarding experience
- Revenue and engagement outcomes

The project focuses on applying real-world Product Analyst and Business Analyst workflows commonly used in SaaS and AI product environments.

---

# Business Problem

AI writing tools often struggle with:
- Poor onboarding experience
- Low-quality prompt generation
- Weak user retention
- Inconsistent output satisfaction

This project evaluates whether guided onboarding and prompt assistance improve:
- Good output rate
- User retention
- Average monthly revenue
- Overall user engagement

---

# Dataset Overview

Synthetic datasets were generated using Python to simulate realistic user behavior across an AI writing assistant platform.

## Tables Used

| Table | Purpose |
|---|---|
| users.csv | User profile and segmentation data |
| sessions.csv | Session activity tracking |
| prompts.csv | Prompt usage and quality scores |
| outputs.csv | AI-generated output feedback |
| experiment_data.csv | A/B testing experiment tracking |
| user_experiment_summary.csv | Post-experiment retention and revenue metrics |

# Dashboard Pages

## 1. Executive Overview

Tracks:
- Total users
- Retention rate
- Good output rate
- Prompt quality
- Feedback distribution
- Prompt type usage

### Key Findings
- Retention remained relatively high at 67.6%
- Good output rate remained comparatively low at 37.9%
- Assignment prompts showed the highest usage volume

---

## 2. User Segmentation Analysis

Analyzes:
- Retention by experience level
- Prompt quality by experience level
- Retention by user type

### Key Findings
- Beginner users showed significantly lower retention
- Higher prompt quality strongly correlated with improved retention
- Student and professional retention remained similar

---

## 3. Experimentation & A/B Testing

Compares:
- Control vs Treatment groups
- Retention after onboarding improvements
- Revenue impact
- Good output rate improvement

### Key Findings
- Guided onboarding significantly improved output quality
- Treatment group retention outperformed control group
- Improved onboarding also increased monthly revenue per user
